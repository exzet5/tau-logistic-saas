# Hospital Equipment Tracker Blueprint

## Overview

This application is designed for a hospital to track the assignment and return of medical equipment. It uses barcode scanning to identify both the equipment and the patient, and it updates a Firebase database to reflect the current status of each item.

## Style and Design

*   **Theme:** The app uses a green and teal color scheme, creating a clean and professional medical look.
*   **Layout:** The layout is simple and intuitive, with a central focus on the main actions: "Take Equipment" and "Return Equipment."
*   **Typography:** The app uses clear, legible fonts for easy readability.
*   **Iconography:** The app uses icons to represent actions and concepts, such as a hospital icon for the logo and a QR code icon for the scanning screen.

## Features

*   **Firebase Integration:** The app is connected to a Firebase project, allowing for real-time data synchronization.
*   **Barcode Scanning:** The app uses the device's camera to scan barcodes, making it easy to identify equipment and patients.
*   **Equipment Tracking:** The app tracks the status of each piece of equipment, including whether it is available or in use.
*   **Patient Assignment:** The app assigns equipment to patients, creating a record of who is using which item.
